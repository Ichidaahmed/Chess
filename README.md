# لعبة شطرنج أونلاين 1v1

لعبة شطرنج متعددة اللاعبين عبر الإنترنت باستخدام Firebase.

## المميزات
- غرف لعب متعددة
- مزامنة فورية
- واجهة متجاوبة
- حركات قانونية فقط

## كيفية الاستخدام
1. افتح [رابط GitHub Pages](https://username.github.io/chess-game-online)
2. أدخل بيانات Firebase
3. أنشئ غرفة أو انضم لواحدة
4. شارك الرابط مع صديقك

## التقنيات المستخدمة
- HTML, CSS, JavaScript
- Chessboard.js
- Chess.js
- Firebase Realtime Database
